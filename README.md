# Mental Wellness Check-In App

Built a Python mood-tracking app that stores and displays wellness entries using JSON. Designed a clean GUI that streamlines user check-ins and provides a simple mood summary.

## Features
- Mood selection with color-coded display
- Daily journal entry
- Saves and displays past entries
- Mood summary analysis (counts of each mood)
- Simple and clean GUI for easy use
